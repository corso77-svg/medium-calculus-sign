# Project structure

- `app/` contains the Android application module.
  - `app/src/main/java/com/example/gesturecapture/` holds Kotlin source files.
  - `app/src/main/res/` contains resources (themes, etc.).
  - `app/src/main/AndroidManifest.xml` defines the app entry point.
- `gradle/` contains Gradle wrapper configuration.
- Root `build.gradle`, `settings.gradle`, and `gradle.properties` configure the build.
